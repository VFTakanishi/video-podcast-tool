# GitHub Pages 公開手順

この無料版は `docs/` フォルダをそのまま公開します。

## 1. GitHub Desktop で反映

1. `GitHub Desktop` を開く
2. 左下の `Summary` に `add free browser-only version` と入れる
3. `Commit to main` を押す
4. `Push origin` を押す

## 2. GitHub で Pages を有効化

1. GitHub でこのリポジトリを開く
2. `Settings` を開く
3. 左の `Pages` を開く
4. `Build and deployment`
5. `Source` を `Deploy from a branch` にする
6. `Branch` を `main`
7. フォルダを `/docs`
8. `Save`

## 3. 公開URL

公開が終わると、数分後に GitHub Pages のURLが出ます。

## 4. 使い方

- URLを開く
- 標準素材を最初に保存する
- 本編動画を選ぶ
- 必要ならジングル位置を調整する
- `MP4を作成する` を押す

## 注意

- 無料版はブラウザだけで処理します
- PC推奨です
- 長い動画は時間がかかります
- 出力は 720p です
